VARIABLES
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
{{SubjectLine}}          ----> Subject replaced             
{{FromEmail}}            ----> From email replaced          
{{FromName}}             ----> From name replaced           
{{Fromdomain}}           ----> From domain replaced                   
{{Domain}}               ----> Domain replaced              
{{HtmlContent}}          ----> HTML Content replaced              
{{PlainContent}}         ----> PLAIN Content replaced             
{{HtmlContent_base64}}   ----> Covert HTML to base64 replaced        
{{PlainContent_base64}}  ----> Covert PLAIN to base64 replaced        
{{HtmlContent_uue}}      ----> Covert HTML to UUENCODE replaced        
{{PlainContent_uue}}     ----> Covert PLAIN to UUENCODE replaced        
{{HtmlContent_qp}}       ----> Covert HTML to quoted-printable replaced        
{{PlainContent_qp}}      ----> Covert PLAIN to quoted-printable  replaced        
{{boundary}}             ----> Replace Boundary with actual value
{{ToEmail}}              ----> Replace To Email
{{MessageId}}            ----> Replaced with Message id
{{ToName}}               ----> Replaced With To email name
{{ToDomain}}             ----> Replaced With To email Domain
((_track_))              ----> Email will Converted into md5 HASH and replaced by this.

                                                                       Content-Transfer-Encoding
-------------------------------------------------------------------------------------------------------------------------------------------------------------

1: 7bit
2: x-uuencode   ---> only for limited ISP: Ex: Comcast
3: quoted-printable
4: base64
5: 8bit
6: binary

EXAMPLE:
Content-Transfer-Encoding: 7bit
Content-Transfer-Encoding: x-uuencode
Content-Transfer-Encoding: quoted-printable
Content-Transfer-Encoding: base64
Content-Transfer-Encoding: 8bit
Content-Transfer-Encoding: binary
                                                                       Charset
------------------------------------------------------------------------------------------------------------------------------------------------------------

1: UTF-8
2: us-ascii
3: windows-1251
4: iso-8859-1

EXAMPLE:
charset=UTF-8
charset=iso-8859-1
charset=us-ascii
charset= windows-1251
